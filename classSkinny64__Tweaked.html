Detailed Description

Abstract base class for SKINNY tweakable block ciphers with 64-bit blocks.

This class is abstract. The caller should instantiate Skinny64_128_Tweaked or Skinny64_192_Tweaked to create a SKINNY block cipher with a tweak and a specific key size.

Reference: https://sites.google.com/site/skinnycipher/

See Also
Skinny64_128_Tweaked, Skinny64_192_Tweaked
